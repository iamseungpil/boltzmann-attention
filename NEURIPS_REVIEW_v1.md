## 1. 논문의 4가지 핵심 기여

### 기여 1: 수학적 증명과 이론적 최적 한계치

8개 회전 기반 KV 캐시 양자화 방법(RoPE, TurboQuant, PolarQuant, CommVQ, FOKVQ, RotorQuant, IsoQuant, SpinQuant)을 Lie 군 프레임워크로 통일하고, 클래스 C(직교 회전 + 독립 스칼라 양자화) 내에서 3축 동시 최적해를 증명.

**축 1 (회전 기저):** Pre-RoPE PCA = 클래스 C 내 MSE 최소 (정리 6.16.3)
- RoPE를 되돌리면 위치 의존성이 소멸, 모든 위치에서 동시 최적
- TurboQuant 대비 이득 = R_aniso(Σ₀) (AM/GM 비율)

**축 2 (양자화기):** MK(마할라노비스 Lloyd-Max) = 어텐션 왜곡 최소 (정리 6.19.8)
- Lloyd-Max 대비 추가 이득 = AM(w)/GM(w) (쿼리 가중치의 AM/GM 비율)
- 균일 양자화기 대비 19~36% MSE 감소 (비트 의존)

**축 3 (토큰 선택):** HEAT 기반 적응적 토큰 관리 (명제 6.20.2)
- 모델별 κ₁, κ₂ 적응적 sink/window 설정
- 고정 규칙(sink=4, window=128) 대비 모델 적응적

**3축 독립성:** 명제 6.18.2에서 3축의 최적화가 상호 독립임을 증명 → 각 축의 이득이 곱셈적으로 결합

**Master Formula:** D*(b, B, Σ) = d · G(b, Σ) · 2^{-2B} (정리 6.1.2)
- 블록 크기 b를 변수로 모든 방법의 왜곡을 단일 공식으로 비교
- AM-GM 계층: G(d) ≤ G(b) ≤ G(1) ≤ G_uniform (따름정리 6.1.4)

### 기여 2: 실험 없이 최적 방법 사전 예측

공분산 행렬 Σ만으로 8개 방법의 상대 성능을 실험 전에 예측하는 3-Level 체계:

| Level | 입력 | 출력 | 비용 | 검증 결과 |
|-------|------|------|------|-----------|
| Level 1 | Σ_K (공분산) | MSE 순서, R_aniso, η(b) | 0.1초 | R² = 1.00 (112/112) |
| Level 2 | Σ + D_actual (실측 MSE 1회) | PPL 예측 | 수 분 | R² = 0.871 (전체), 0.996 (3-bit) |
| Level 3 | Σ + D_actual + gap(p) | NIAH 위치별 정확도 | 수 분 | 미검증 |

알고리즘 6.12.1: 8개 방법 → 2-3개 후보로 축소 (Stage 1: 0.1초) → MSE 실측 + PPL 캘리브레이션 (Stage 2: 수 분)

### 기여 3: 예측값과 실측값의 오차 범위 내 일치

| 검증 항목 | 결과 |
|-----------|------|
| MSE 순서 (Level 1) | 112/112 헤드 100% 성립 |
| PPL 예측 (Level 2, 2-point) | 4-bit 오차 ≤2.3%, 3-bit 오차 ≤6.6% |
| Pre-RoPE PCA > TurboQuant | 모든 비트에서 MSE 최저, 2-bit PPL 14.4% 개선 |
| Post-RoPE PCA가 2-bit에서 TurboQuant에 패배 | MSE 1.43배 패배 확인 (따름정리 6.16.4(d)) |
| MSE 순위 = PPL 순위 | 100% 일치 |

### 기여 4: 클래스 C의 완전 최적해 (3축 동시)

| 축 | 최적해 (본 논문) | KVTC | TurboQuant | SpinQuant | PolarQuant |
|----|-----------------|------|------------|-----------|------------|
| 축 1: 회전 | 헤드별 Pre-RoPE PCA | 공유 PCA (비최적) | 랜덤 (비최적) | 학습 (보장 없음) | SO(2) 한정 |
| 축 2: 양자화기 | MK Lloyd-Max | 균일 스칼라 (비최적) | 균일 (비최적) | 균일 (비최적) | ad-hoc |
| 축 3: 토큰 선택 | HEAT 적응적 | 고정 규칙 (비최적) | 없음 | 없음 | 없음 |
| **최적 축 수** | **3/3** | **1/3 (부분)** | **0/3** | **0/3** | **0/3** |

**핵심 주장:** 기존 모든 방법은 최소 1개 축에서 비최적이며, 본 논문이 3축 모두의 최적해를 처음으로 증명하고 결합.

## 2. 기존 논문 분석 (Novelty 근거)

8편의 기존 논문을 직접 분석한 결과, **모든 논문이 자기 방법만 ad-hoc으로 최적화**:

| 논문 | Lie 군 이론 | 통일 프레임워크 | 일반적 최적성 증명 | RoPE 토러스 인식 | 방법 간 이론적 비교 |
|------|:-----------:|:---------------:|:------------------:|:----------------:|:-------------------:|
| PolarQuant (NeurIPS 2025) | 없음 | 없음 | 없음 | 없음 | 없음 |
| TurboQuant (ICLR 2026) | 없음 | 없음 | 자체 bound만 | 없음 | 없음 |
| CommVQ (ICML 2025) | 없음 | 없음 | 없음 | 없음 | 없음 |
| SpinQuant (ICLR 2025) | 없음 | 없음 | 없음 | 없음 | 없음 |
| KVTC (ICLR 2026) | 없음 | 없음 | DP 비트 할당만 | 없음 | 없음 |
| IsoQuant (arXiv 2026) | 부분적 | 없음 | 없음 | 없음 | 없음 |
| QuIP# (ICML 2024) | 없음 | 없음 | 자체 선택만 | 없음 | 없음 |
| KVQuant (NeurIPS 2024) | 없음 | 없음 | 없음 | 없음 | 없음 |

**구체적 gap:**
- PolarQuant: 극좌표를 쓰면서 이것이 SO(2) 궤도 분해인 줄 모름
- TurboQuant: 랜덤 회전을 쓰면서 Haar 측도/등방화의 군론적 의미를 모름
- CommVQ: 교환 조건을 2×2 행렬 성질로만 처리, 중심화군 개념 없음
- SpinQuant: Cayley SGD로 so(d)를 암묵적으로 사용하면서 명시적 인식 없음
- KVTC: Pre-RoPE PCA를 쓰면서 왜 최적인지 증명하지 않음, 공유 PCA 사용

### 4.2 다중 모델 실험 (Reproducibility 3 → 6)

**필수 모델 3개:**

| 모델 | 크기 | GQA 그룹 | 선택 이유 |
|------|------|----------|-----------|
| Qwen2.5-7B | 7B | G=7 | 기존 데이터 있음, R_aniso=4.07 |
| Llama-3-8B | 8B | G=4 | KVTC 주요 모델과 직접 비교 가능, R_aniso 확인 필요 |
| Mistral-7B-v0.3 | 7B | G=4 | R_aniso=80.89 (극단적), 이론 검증에 최적 |

**각 모델에서 검증할 항목:**

```
A. Level 1 검증 (MSE 순서)
   - 모든 헤드에서 G(d) ≤ G(b) ≤ G(1) ≤ G_uniform 성립 여부
   - R_aniso, η(b) 실측값 vs Table 1 예측값

B. Level 2 검증 (PPL 예측)
   - 비교 방법: No rotation, TurboQuant, Post-RoPE PCA+WF, Pre-RoPE PCA+WF
   - 비트: 2, 3, 4 bit
   - ln(PPL/fp16) ≈ c · D_actual의 R² 값

C. 정리 6.16.3 검증 (Pre-RoPE PCA 최적성)
   - Pre-RoPE PCA가 모든 비트에서 MSE 최저인지
   - TurboQuant 대비 이득이 R_aniso에 비례하는지
   - Post-RoPE PCA가 2-bit에서 TurboQuant에 패배하는지
```

**실험 우선순위:**
1. [최우선] Llama-3-8B에서 Pre-RoPE PCA vs TurboQuant MSE/PPL 비교 (KVTC와 동일 모델)
2. [높음] Mistral-7B에서 R_aniso=80.89 검증 (극단적 이방성에서 이론 정합도)
3. [중간] 3모델 Level 1 검증 (MSE 순서 100% 성립 확인)

### 4.3 Downstream 벤치마크 (Significance 확장)

**필수 벤치마크:**

| 벤치마크 | 유형 | 역할 | KVTC 대응 |
|----------|------|------|-----------|
| PPL (WikiText-2) | 평균 품질 | Level 2 검증 | ✓ |
| NIAH (4K, 8K, 16K) | 위치별 검색 | HEAT(축3) 차별화, Level 3 검증 | ✓ (70B만) |
| GSM8K (8-shot CoT) | 추론 | 단문맥 품질 확인 | ✓ |

**벤치마크 실험 매트릭스:**

```
모델 3개 × 방법 4개 × 비트 3개 = 36 실험

방법:
  1. No rotation (baseline)
  2. TurboQuant (random SO(d))
  3. Pre-RoPE PCA + WF (축1 최적)
  4. Pre-RoPE PCA + MK + WF (축1+2 최적)

비트: 2, 3, 4 bit

측정:
  - MSE (헤드별 평균)
```

**NIAH 실험 상세:**

- 8B 모델에서 KVTC의 고정 규칙(sink=4, window=128)이 부족할 수 있는 조건이 차별화 기회

## 7. 핵심 리스크와 대응

### 리스크 1: Pre-RoPE PCA가 Llama/Mistral에서 Qwen과 다른 결과
- **대응:** 이론은 모델에 무관하게 성립 (정리 6.16.3). 실측 이득의 크기가 다를 수 있으나 순서는 동일해야 함. 만약 순서가 깨지면 가우시안 가정의 위반 정도를 분석.

### 리스크 2: Mistral R_aniso=80.89의 수치적 불안정성
- **대응:** Table 1에서 범위가 1.7~4805.3으로 극단적. 특정 레이어/헤드의 수치적 특이값 확인 필요. 이상치 헤드를 제외한 robust 통계량도 보고.

### 리스크 3: NIAH에서 HEAT가 고정 규칙과 차이가 없을 경우
- **대응:** Section 6.22의 시나리오 (ii) — PPL 우위에 집중하고, 축 3의 기여를 "동등 + 이론적 정당화"로 전환.

### 리스크 4: KVTC 재현이 어려울 경우
- **대응:** KVTC 논문의 보고 수치를 인용하고, 동일 조건(모델, 비트)에서 본 논문 방법의 결과를 직접 비교. 공정성을 위해 "KVTC 보고값 인용"임을 명시.

## 8. 최종 논문 포지셔닝 전략

**제목 안:** "Optimal KV Cache Quantization via Lie Group Theory: A Unified Framework with Provable Optimality on Three Independent Axes"

**한 줄 요약:** RoPE의 토러스 대칭성을 기반으로 8개 회전 기반 KV 캐시 양자화 방법을 Lie 군 프레임워크로 통일하고, 회전 기저·양자화기·토큰 선택 3축 각각의 이론적 최적해를 증명하여, 기존 모든 방법(KVTC 포함)이 도달하지 못한 클래스 C의 완전 최적해를 제시한다.

**메인 메시지 (리뷰어에게 전달할 핵심):**
1. 기존 8편이 각각 ad-hoc으로 최적화하던 것을, 처음으로 하나의 수학적 언어로 통일
2. 통일 후 이론적 최적 한계를 증명하여, 어떤 방법이 얼마나 비최적인지 정량화
3. 수학적 예측이 실험 없이도 최적 방법을 추천할 수 있음을 입증
4. 3축 동시 최적을 달성하여, KVTC를 포함한 기존 모든 방법을 이론적으로 상회
