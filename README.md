# boltzmann-attention

Experiment code, configs, results, and report sources for the Boltzmann Attention / FOKVQ workstream.

## Included

- `scripts/`: experiment runners and benchmark code
- `results/`: JSON outputs and experiment logs
- `reports/`: Markdown/LaTeX report sources and generated figures
- `*.yaml`, `.amltconfig`: AMLT and launch configs

## Excluded

- `.docx` planning drafts and other local binary documents
- `.zip` archives
- generated PDF / LaTeX build artifacts
- `paper/`: separate nested paper repository
