\begin{abstract}
9개 실험(1-1 -- 3-1)의 결과를 종합 보고한다.
Boltzmann Level 1 실험 3개에서 유효 온도 $T_\mathrm{eff}$의 점진적 냉각은 표준 MHA(GPT-2 Medium)에서 확인되었으나($\rho = -0.657$, $p < 0.001$), GQA(Qwen2.5-3B)에서는 성립하지 않았다($\rho = +0.108$, $p = 0.530$).
비열 $C(q)$는 키 노름 $\|k\|^2$와 39.8\% 불일치를 보여 독립적 물리량임이 확인되었으나, PPL 개선에는 실패하였다.
FFN 주파수 선택성 가설은 기각되었다($\Delta H = -0.122$, $p < 10^{-10}$).
FOKVQ Phase 2에서 Facet K-space 분리도는 코퍼스 간 유의한 차이를 보였고(gap = 0.408, $p = 1.6 \times 10^{-64}$), PCA 기반 비균일 비트 할당이 LDA 대비 K MSE에서 26--30배 우수하였다.
실험 2-3(query-dependent vs static 할당)은 두 독립 구현이 상충하는 결과를 산출하여 \textbf{Inconclusive}로 판정하였다.
AFOD 초기화는 PCA 대비 유의한 개선을 보이지 못하였다($p > 0.4$).
Phase 3 Lie Group 검증에서 SO($d$) 수치 최적화는 PCA 대비 4.3배 열위로, PCA 기저의 사실상 최적성이 확인되었다($p = 0.011$).
종합하면, Boltzmann 프레임워크는 기술적(descriptive) 가치를 가지나 규범적(prescriptive) 양자화 설계에는 부적합하며, FOKVQ의 정적 PCA 기반 facet-aware 접근이 양자화 성능의 핵심 동력임이 재확인되었다.
\end{abstract}

\section{Boltzmann Level 1}

이 절은 소프트맥스 어텐션과 볼츠만 분포의 형식적 동일성이 KV 캐시 양자화에 실용적 가치를 제공하는지를 3개 실험으로 검증한다.

\subsection{실험 1-1: $T_\mathrm{eff}$ 게이트 테스트}

\subsubsection{목적 및 설정}

이 실험은 Boltzmann 프레임워크의 핵심 예측인 ``점진적 냉각(progressive cooling)''을 검증한다.
소프트맥스 어텐션을 볼츠만 분포로 해석하면, 심층 레이어로 갈수록 유효 온도 $T_\mathrm{eff}^{(l)} = S^{(l)} / \log n$이 단조 감소해야 한다.
$S^{(l)} = -\sum_j \alpha_j \log \alpha_j$는 어텐션 엔트로피이다.
Forward hook으로 각 레이어에서 $Q$, $K$를 추출한 후 $\alpha = \mathrm{softmax}(QK^\top / \sqrt{d})$를 직접 계산하여 $T_\mathrm{eff}$를 측정하였다.

\subsection{실험 1-3: FFN 주파수 선택성}

\subsubsection{목적 및 설정}

이 실험은 FFN이 어텐션 에너지 스펙트럼에서 주파수 선택성 필터 역할을 하는지 검증한다.
DFT를 통해 $Q$, $K$의 주파수 성분별 에너지 기여도 $p(\omega)$를 계산하고, FFN 포함/제거 조건에서 스펙트럼 엔트로피 $H(\omega) = -\sum p(\omega) \log p(\omega)$를 비교하였다.
FFN 제거는 forward hook으로 FFN 출력을 영벡터로 대체하여 구현하였다.
모델: GPT-2 Medium(24L/16H/$d$=64), 48개 샘플.

\subsubsection{결과}

FFN은 스펙트럼 엔트로피를 \textbf{증가}시켰으며, 이는 가설의 반대 방향이다.

\section{FOKVQ Phase 2}

이 절은 FOKVQ의 facet-aware 비균일 비트 할당의 이론적 기반을 4개 실험으로 검증한다.

\subsection{실험 2-1: Facet K-space 분리도}

\subsection{실험 2-3: Query-Dependent vs Static 비트 할당}

\subsubsection{목적 및 설정}

이 실험은 PCA 기반 비트 할당이 query에 따라 동적으로 조정되어야 하는지, 아니면 정적(static) 할당으로 충분한지를 검증한다.
Query-dependent 할당은 각 query $q$에 대해 $\mathrm{Var}_{k}[q^\top k_j]$를 차원별로 계산하여 분산이 높은 차원에 더 많은 비트를 할당한다.
정적 할당은 캘리브레이션 데이터 전체에서 한 번 계산한 PCA 분산 순위를 모든 query에 동일하게 적용한다.

\label{tab:exp23_setup}
\begin{tabular}{ll}
\toprule
\textbf{} & \textbf{Value} \\
\midrule
Model & GPT-2 Medium (24L/16H/$d$=64) \\
Samples & 60 \\
Metrics & Spearman $\rho$(static, q-dep), L1 distance, Q-Q $\rho$ \\
\bottomrule
\end{tabular}
\end{table}

사전 등록 기준: (1) static vs.\ query-dependent 순위 상관 $\rho < 0.9$이면 query-dependent 할당이 유의미하게 다른 순위를 생성, (2) query 간 순위 상관 $\rho < 0.9$이면 query마다 할당이 변동.

\subsubsection{결과}

판정: \textcolor{inconclusive}{Inconclusive}.

두 가지 독립 구현이 상충하는 결과를 산출하였다.
구현 A(seq=128)에서는 24개 레이어 중 23개에서 query-dependent 분산 순위가 상수로 퇴화하여 Spearman $\rho$가 정의 불가(NaN)였으며, \texttt{allocation\_differs = false}로 판정되었다.
구현 B(seq=256)에서는 \texttt{allocation\_differs = true}, 전체 평균 $\rho = 0.698$로 query-dependent 할당이 정적 할당과 유의하게 다르다는 결과를 보였다.

이 불일치의 원인은 시퀀스 길이, threshold 방식, 비등방성 처리 등 구현 세부사항에 결과가 민감하기 때문으로 판단된다.
어느 구현의 결과가 정확한지를 현재 데이터만으로 결정할 수 없으므로, 특정 수치를 인용하지 않는다.

\subsubsection{시사점}

이 실험은 표준화된 프로토콜(시퀀스 길이, 할당 threshold, 비등방성 처리 방식 고정)로 재실험이 필요하다.
K-space의 극단적 비등방성(PC1이 분산의 98\% 이상을 설명하는 상황)에서 median 기반 할당은 사실상 단일 버킷으로 축소될 수 있으며, 이 경우 query-dependent 할당과 정적 할당의 구별이 본질적으로 불가능해진다.
``정적 할당이 충분하다''와 ``query-dependent 할당이 다르다''는 두 해석이 모두 열려 있으며, 비등방성이 덜한 모델이나 초기 레이어에서의 재검증이 필요하다.

둘째, 64개 Givens 평면으로는 $d=64$ 차원 공간의 자유도 $d(d-1)/2 = 2016$을 충분히 탐색하지 못하며, 최적화 자체가 고차원 비볼록 문제에서 비효율적이다.
PCA의 폐쇄형 해는 계산 효율성과 최적성 모두에서 수치 최적화를 압도하며, FOKVQ 설계의 핵심 선택이 최적에 근접함을 지지한다.

\section{Cross-experiment Synthesis}

이 절은 9개 실험 결과를 통합 분석하여, Boltzmann 프레임워크와 FOKVQ 접근 간의 연결점과 경계를 규명한다.

\subsection{확인된 것과 기각된 것}

\paragraph{확인된 것.}
\begin{enumerate}[nosep]
  \item 점진적 냉각($T_\mathrm{eff}$ 단조 감소)이 표준 MHA에서 실험적으로 확인되었다($\rho = -0.657$, $p < 0.001$).
  \item GQA가 점진적 냉각을 파괴한다는 새로운 발견이 이루어졌다(inter-group gap = 0.506).
  \item $C(q)$는 $\|k\|^2$와 독립적인 새로운 물리량이다(39.8\% disagreement, $\rho = 0.275$).
  \item K-space의 PCA facet 구조는 코퍼스 간에 유의하게 분리된다(gap = 0.408, $p = 1.64 \times 10^{-64}$).
  \item 양자화에서 분산 최대 방향(PCA)이 판별 방향(LDA)보다 우수하다(K MSE 26--30배).
  \item AFOD 초기화는 PCA와 통계적으로 구별되지 않는 성능을 보인다($p > 0.4$, 3개 비트 수준).
  \item PCA 기저가 SO($d$) 수치 최적화 대비 4.3배 우수하여, 사실상 최적의 회전임이 확인되었다($p = 0.011$).
\end{enumerate}

\paragraph{기각된 것.}
\begin{enumerate}[nosep]
  \item FFN이 주파수 선택성 필터 역할을 한다는 가설($\Delta H = -0.122$, 24/24 레이어에서 반대 방향).
  \item $C(q)$ 기반 단순 양자화가 $\|k\|^2$ 기반 방법을 능가한다는 기대(Heat wins: 0).
  \item GQA 모델에서 볼츠만 점진적 냉각이 성립한다는 전제($\rho = +0.108$, $p = 0.530$).
  \item AFOD 초기화가 PCA 대비 유의한 개선을 제공한다는 기대(개선폭 $< 1\%$, $p > 0.4$).
  \item SO($d$) 수치 최적화가 PCA보다 나은 회전을 발견할 수 있다는 기대(PCA 4.3배 우수, $p = 0.011$).
\end{enumerate}

\paragraph{미결 사항.}
실험 2-3(query-dependent vs static 할당)은 두 구현이 상충하여 결론을 내릴 수 없다.
표준화된 프로토콜로 재실험하여 정적 할당의 충분성 여부를 확정해야 한다.

\subsection{Boltzmann + FOKVQ 연결점}

Boltzmann Level 1과 FOKVQ Phase 2는 소프트맥스 어텐션의 서로 다른 측면을 조명한다.
Boltzmann 프레임워크는 레이어/헤드 수준의 거시적(macroscopic) 열역학적 특성을 포착하며, FOKVQ는 차원 수준의 미시적(microscopic) 분산 구조를 활용한다.

\paragraph{확인된 연결.}
$C(q)$(실험 1-2)와 Facet Gain 간의 상관($\rho = -0.63$, $p = 0.001$)은 Boltzmann 양이 FOKVQ 성능을 예측하는 간접 지표로 역할할 수 있음을 보여준다\footnote{Source: FOKVQ\_FINAL\_REPORT.docx Section 4D.2. .docx 형식이므로 기계 검증 미완료.}.
$C(q)$가 높은(민감도가 높은) 레이어에서 facet-aware 양자화의 이득이 작고, 낮은 레이어에서 이득이 큰 패턴이다.

\paragraph{단절된 지점.}
$T_\mathrm{eff}$와 Facet Gain 간의 상관은 무상관이다($\rho = +0.30$, $p = 0.16$)\footnote{동일 출처(FOKVQ\_FINAL\_REPORT.docx). 기계 검증 미완료.}.
어텐션의 거시적 집중도는 양자화 오류의 미시적 차원 구조와 직접적 관계가 없다.
FFN 주파수 선택성 가설(실험 1-3)의 기각은 Boltzmann 프레임워크의 적용 범위가 어텐션 메커니즘에 한정되며 FFN까지 확장되지 않음을 의미한다.

\paragraph{열려 있는 문제.}
$C(q)$와 $\|k\|^2$의 결합 기준이 개별 기준을 능가할 수 있는지는 후속 실험으로 검증해야 한다.
실험 3-1에서 Givens 매개변수 64개는 자유도 2016의 3.2\%만 탐색하므로, 더 많은 Givens 평면 또는 gradient 기반 최적화가 PCA에 근접할 수 있는지는 열린 질문이다.
